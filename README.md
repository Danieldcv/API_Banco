# API em C# para gestão de contas bancárias
## Requisitos:
- Endpoint para cadastrar nome, e-mail, senha e gerar o número da conta
aleatoriamente;
- Endpoint para realizar login com e-mail e senha;
- Endpoint para obter o saldo da conta;
- Endpoint para obter o extrato da conta;
- Endpoint para transferir valores entre contas.
